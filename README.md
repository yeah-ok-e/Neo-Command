# NEO Command

NEO Command is a mobile-first personal advancement operating system: a simple command center for work, goals, money, routines, life areas, and next moves.

It is designed for anyone trying to organize, stabilize, and advance: cashier, student, corporate worker, founder, creator, freelancer, parent, athlete, or operator in transition.

## What is included

- Animated NEO load page
- Under-60-second intake form
- Optional sensitive data banner
- LocalStorage persistence
- Placeholder-safe dashboard
- Work / main occupation framing
- Money snapshot
- Goals and routines
- AI Next Move prototype panel
- Admin preview structure
- Mobile-first premium black UI

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy on Vercel

1. Create a new GitHub repository named `neo-command`.
2. Upload this project into the repo.
3. Go to Vercel and import the repo.
4. Use framework preset: Vite.
5. Build command: `npm run build`.
6. Output directory: `dist`.
7. Deploy.
